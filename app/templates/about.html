SciTools Radar

Plataforma de monitoreo automatizado de herramientas de investigación científica, impulsada por agentes de inteligencia artificial.

¿Qué es SciTools Radar?

SciTools Radar es un sistema inteligente que rastrea, clasifica y reporta herramientas digitales relevantes para la investigación académica y científica. El sistema combina un pipeline de agentes de IA con curaduría humana para mantener un inventario actualizado y generar entradas de blog informativas.

Desarrollado por Smart Centrum, el radar opera de forma autónoma con actualizaciones periódicas, manteniendo a la comunidad informada sobre nuevas herramientas y cambios relevantes en las ya catalogadas.

Pipeline de Agentes IA

El proceso de generación de contenido utiliza un pipeline de 5 etapas, cada una con un modelo especializado:

1
Investigador
claude-sonnet-4

Busca nuevas herramientas y novedades vía Tavily API

2
Clasificador
claude-sonnet-4

Categoriza y asigna campos científicos

3
Redactor
claude-opus-4

Genera el editorial profesional en formato JSON

4
Creador
Flask/SQLAlchemy

Persiste la entrada en la base de datos

5
Evaluador
claude-sonnet-4

Evalúa la calidad (0–10) y proporciona retroalimentación

Automatización

El pipeline se ejecuta de forma automática según un calendario configurado. Cada ejecución investiga nuevas herramientas, detecta actualizaciones en las existentes, genera un editorial profesional y evalúa la calidad del contenido.

Martes y Jueves a las 07:00 UTC Dos ejecuciones semanales para mantener el radar actualizado sin sobrecargar los recursos de la plataforma.
